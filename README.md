## ⚙️ Qurulma və İşə Salma

Layihəni local komputerdə işə salamq üçün aşağıdakı addımları et:

```bash
# 1. Reponu klonla
git clone https://github.com/ismetcahangirov/codeworld.git

# 2. Layihə qovluğuna keç
cd codeworld

# 3. Lazımi paketləri yüklə
npm i

# 4. Layihəni işə sal
npm run dev
```
